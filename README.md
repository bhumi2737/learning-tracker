# 📚 Learning Tracker Web Application

## 📌 Project Overview

The **Learning Tracker Web Application** is a full-stack web project built using **Node.js, Express.js, and a simple frontend (HTML, CSS, JavaScript)**.
The purpose of this project is to help users **track their learning tasks, manage courses, and monitor progress**.

Instead of using a database, the application stores data in **JSON files**, which keeps the project simple and aligned with backend concepts taught in class.

The project demonstrates the implementation of **REST APIs and CRUD operations**.

---

# 🚀 Features

* 📊 Dashboard to view learning statistics
* ✅ Add new learning tasks
* 📋 View all tasks
* ✏ Update existing tasks
* 🗑 Delete tasks
* 👤 Associate tasks with a user
* 📚 Courses page showing learning progress
* 💾 Data stored using JSON files (no database)
